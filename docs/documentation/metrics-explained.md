# Metrics Explained
This is the calculation schema for the relevant metrics:
![Calculation schema for all the metrics](/calculation-schema.png)

Here are all the metrics that can be calculated using the USM Calculator Webservice:

## Overview
| Metric | Name | 
| -------- | ------|
| [WUP](metrics/wup) | Weighted Urban Proliferation |
| [DIS](metrics/dis) | Degree of Urban Dispersion |
| [PBA](metrics/pba) | Proportion of Built-up Areas |
| [LUP](metrics/lup) | Land Uptake per person |
| [UD](metrics/ud) | Utilization Density |
| [UP](metrics/up) | Urban Permeation |
| [WUP<sub>b</sub>](metrics/wupb) | Weighted Urban Proliferation of the settleable part of the study area |
| [WSPC](metrics/wspc) | Weighted Sprawl per Capita |
| [TS](metrics/ts) | Total Sprawl |

## PBA, DIS, LUP in reality
In the real world these three values can be visualized using the image below.

![How PBA, DIS and LUP occur in the real world](/pba-dis-lup.png)

The green-red arrow shows the spread of the values. Green are smaller values for the indicators, resulting in a smaller final WUP Result.
If these values are higher (red arrow direction), they contribute to the urban sprawl (higher WUP).
